The javascript performs the calculations while the HTML handles the user interface of my quiz. The first part of my javascript defines the variables based of the id's of my elements from my HTML. I then added an event listener so that when the user "clicks" the button for the answer they choose, the function would perform the slew of if and if else calculations which pertain to the particular operator employed by the user. When the user clicks on there answer, the event listener function alerts them through the browser.
