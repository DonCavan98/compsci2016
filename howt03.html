The HTML for the Calculator consisted of a selection menu with a total of 6 operators for the user to choose from. The selection menu was used in conjunction with two inputs which the user would choose which numbers to calculate with respect to the particular operator. These options were in one line with the selection menu in between the two inputs to simplify the users' experience. Underneath was a div which would display the answer of each particular operation.
How to make a Homepage Explaination 
            
var num1input = document.getElementById("num1");
var operator = document.getElementById("operator");
var num2input = document.getElementById("num2");
var calculate = document.getElementById("calculate");
var display = document.getElementById("display");

calculate.addEventListener("click", function(){
  var num1 = num1input.value;
  var num2 = num2input.value;
 if(operator.value === "+"){
    var answera = +num1 + +num2;
    display.innerHTML = answera;
  }
  else if(operator.value === "-"){
    var answerb = num1 - num2; 
    display.innerHTML = answerb;
  }
   else if(operator.value === "*"){
    var answerc = num1 * num2; 
    display.innerHTML = answerc;
  }
   else if(operator.value === "/"){
    var answerd = num1 / num2; 
    display.innerHTML = answerd;
  }
  else if(operator.value === "^"){
    var answere = Math.pow(num1,num2);
    display.innerHTML = answere;
  }
  else if(operator.value === "sqrt"){
  var answerf = Math.sqrt(num2)
  display.innerHTML = answerf;
  }
});
The javascript performs the calculations while the HTML handles the user interface of my calculator. The first part of my javascript defines the variables based of the id's of my elements from my HTML. I then added an event listener so that when the user "clicks" the calculate button, the function would perform the slew of if and if else calculations which pertain to the particular operator employed by the user.
