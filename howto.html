I made the my hompage within "index.html" where I included multiple links sourced to separate html pages. This homepage would provide the user a brief intro as to the content of each section of my webpage. I used headers and paragraphs for labels which those who visit the site would read, along with css styling for aesthetic purposes such as coloring and font-type. I chose the color aesthetic because a red hue and black is my favorite colorway. I chose the font family of Helvetica, Arial, Sans-serif, because it is very easy to read and is very soft, so I thought it would mesh well with the coloring of the website.
